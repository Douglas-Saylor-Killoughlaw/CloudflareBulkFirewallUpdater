Cloudflare Bulk Firewall Updater
=============

This is a simple script to bulk pdate you firewall rules on Cloudflare. 

0. Drop the files onto a web server
0. Browse to cloudflare.php
0. Fill in the form
0. Done. Time for tea and biscuits.
